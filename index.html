DESENVOLVENDO .

Como desenvolvedor Full Stack, meu trabalho consiste em unir lógica e criatividade, crio soluções técnicas e funcionais, combinadas com elementos visuais atraentes. Minha abordagem equilibra eficiência lógica e inovação criativa em cada projeto.

MINHAS TECNOLOGIAS

JAVA

Me aprimorando no uso do Java em conjunto com o Spring Boot, dada sua versatilidade e a API JDBC para gerenciamento de dados. Minha atenção se volta ao desenvolvimento de web apps, apps, soluções em Big Data, e aplicações financeiras, onde o Spring Boot, com sua integração ao Spring Data, simplifica o acesso a bancos de dados e a gestão de transações.

PHP

Eu escolho o PHP com o Laravel para desenvolvimento web. Devido à eficácia do PHP na geração de conteúdo dinâmico e às vantagens adicionais que o Laravel oferece. O Laravel fornece um conjunto abrangente de ferramentas robustas que tornam o desenvolvimento mais ágil e eficiente, resultando em bases de código bem estruturadas e de fácil manutenção.

Web

JavaScript, HTML e CSS são essenciais no desenvolvimento web, trabalhando juntos para criar experiências boas e visualmente agradáveis. Uso o CSS com o Bootstrap essa combinação em projetos que exigem responsividade e funcionalidades modernas, economizando tempo de desenvolvimento e keyframes para animações especificas.
